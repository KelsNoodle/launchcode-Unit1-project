# launchcode-Unit1-project
Bird trading card site

Purpose: This website is designed to create a simple card pulling game for students and
        teachers to educate them about birds. It was created as a free solution to gacha 
        mechanics while also providing educational material. 

Getting Started: This repository uses the following: React, React Router, React Router DOM, Vite, Javascript, and HTML. 

After cloning, open a terminal at /unit1-project and install the dependencies:
    npm install

In order to demo the site without deploying, in the same location:
    npm run dev

File Structure:

├── public/            # Location of all images
├── src/
│   └── components/    # Reusable components that populate the site and provide its functions
│   └── data/
│       └── cards.json # database that references all of the card images and is used when drawing cards    
├── app.css            # Primary CSS styling for site
├── app.jsx            # Primary file for routing and conditional rendering
├── main.jsx           # Parent file for Browser Router and app.jsx
├── index.html         # Homepage (main entry point) - used to deploy to github
├── package-lock.json  # generated by npm - locks dependency versions in order to maintain stability
├── package.json       # dependency list that npm uses to install all modules necessary
├── vite.config.js     # created after installing dependencies (vite)
└── README.md          # Project documentation (this file)
